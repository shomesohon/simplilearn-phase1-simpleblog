Simplilearn Phase 1: A simple blog


Used Library
1. Bootstrap
2. JQuery
3. Fontawesome


In this project I have created 2 portal 1 for the Admin for the company and another for Users.

ADMIN
Create a new blog with validation using javascript
list of blogs in one place to manage
Update the existing blog

USER
This project is only a frontend project.
A simple home page for user with a sliding banner and some recent blogs
Listed category for better searching experience
Detailed blog for user

I have not attached bootstrap file in the sourche code. I have 'git ignored' the file.
I have used CDN for JQuery and Fontawesome
